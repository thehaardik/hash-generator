# 🔑 Hash Generator

Generate MD5, SHA-1, SHA-256, SHA-512 hashes from any text. Instant, secure, runs locally.

## 🚀 Features

- **Multiple Algorithms** — MD5, SHA-1, SHA-256, SHA-512, SHA-3
- **Instant Results** — Hashes generated as you type
- **Uppercase/Lowercase** — Toggle output format
- **Copy to Clipboard** — One-click copy
- **No Server** — Runs entirely in browser

## 🎯 Why This?

Hash generators are often ad-filled websites. This one is clean, fast, and private.

## 🛠 Tech Stack

- Next.js 14
- React
- Tailwind CSS
- shadcn/ui
- Web Crypto API

## 📦 Setup

```bash
git clone https://github.com/thehaardik/hash-generator.git
cd hash-generator
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
